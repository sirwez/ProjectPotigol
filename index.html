Potigol é...

Uma linguagem moderna (funcional) para aprender a programar.


Exemplo de código:

 
  #programa para calcular área e perímetro de um quadrado


        tipo Quadrado
            lado: Real
            area() = lado * lado
            perimetro() = 4 * lado
        fim

        q1 = Quadrado(10)
        escreva q1.area
        escreva q1.perimetro

Quer saber mais sobre a linguagem?

Gitpod é uma IDE online para GitHub e GitLab que lança ambientes de desenvolvimento prontos para codificar para qualquer projeto com um único clique. Acesse o ambiente pré-configurado para resolução de questões do URI Online Judge totalmente na linguagem Potigol.

IDE Online

Atom é um editor de texto de código aberto disponível para as plataformas Linux, macOS e Microsoft Windows, desenvolvido pelo GitHub sob a licença MIT. O Editor oferece suporte para diversas linguagens, incluindo Potigol.
